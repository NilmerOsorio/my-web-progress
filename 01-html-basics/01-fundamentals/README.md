# 📝 HTML Basics Exercise

A simple practice project focused on understanding the core foundation of web development.

---

## 🚀 What I Learned
In this exercise, I learned the fundamental structure of an HTML5 document, including:
* The importance of the `<!DOCTYPE html>` declaration.
* The roles of the `<html>`, `<head>`, and `<body>` tags.
* How meta tags and page titles function within the document configuration.

---

## 🛠️ Technologies Used
* **HTML5** — Core structure and semantics.

# HTML Forms Practice - Part 1

A simple, semantic HTML registration form built as a practical exercise to reinforce core web development concepts. This project focuses on understanding user input, semantic structuring, and accessible form controls.

## 🎯 Objectives & Learnings

This exercise covers the foundational concepts of HTML5 forms:
* **`<form>` Structure:** Understanding how to enclose user input elements properly.
* **Form Layout:** Using semantic paragraph tags (`<p>`) to structure and space form fields vertically without relying on excessive break tags.
* **Accessibility & Labels:** Linking `<label>` elements to `<input>` and `<textarea>` fields using the `for` and `id` matching attributes.
* **Input Types:** Implementing standard `text` inputs for short answers and `<textarea>` elements for multi-line user feedback.
* **Form Submission:** Utilizing a dedicated `<button type="submit">` control to handle form processing.

## 🛠️ Built With

* **HTML5:** Semantic markup and basic form structures.

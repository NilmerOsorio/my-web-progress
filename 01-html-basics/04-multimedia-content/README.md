# My Media Hub 🎬🎵📸

A responsive and semantic HTML5 practice project built to demonstrate the implementation of native and embedded multimedia elements in web development.

---

## 🔍 Project Overview
 The main goal was to move beyond text-based HTML and master the integration of rich media—including self-hosted images, native video/audio streams, and external third-party embeds (iframes).

---

## ✨ Key Features
* **Image Gallery:** Showcases remote high-resolution landscape wallpapers with customized dynamic dimensions.
* **Multi-Format Video Playback:** Demonstrates dual implementation methodologies for streaming video files (`.mp4`) using both direct syntax and child `<source>` nesting.
* **YouTube Integration:** Features an interactive `<iframe>` integration utilizing specialized embed restrictions to allow full-screen execution and hardware acceleration (Featuring the ultimate *Rickroll* classic).
* **Accessible Audio Infrastructure:** Leverages native HTML5 `<audio>` controllers complete with backwards-compatible fallback textual scripts for legacy browsers.

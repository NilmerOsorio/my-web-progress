# 📝 Biography of Mark Zuckerberg (HTML Text Tags Exercise)

This project is a foundational web development exercise focused on using standard HTML text formatting tags to build a structured biography of **Mark Zuckerberg**.

---

## 🚀 Project Overview

The goal of this project was to practice basic text styling, spacing, and structural hierarchies using pure HTML without any external CSS styling. 

### 🔑 Key Concepts Covered
* **Headings (`<h1>`, `<h2>`, `<h3>`):** Creating a multi-level visual hierarchy for sections like Early Life, Facebook's Founding, and Meta's Leadership.
* **Inline Text Styling (`<i>`, `<b>`, `<u>`):** Combining tags to italicize, bold, and underline headers and key entities (e.g., *<u>Personal Information</u>*, ***"ZuckNet"***).
* **Spacing (`<br>`):** Utilizing manual line breaks to separate text blocks and force text wraps within paragraphs.
* **Quotes:** Implementing a prominent concluding inspirational quote from Zuckerberg at the bottom of the page.

---

## 🛠️ Built With

* **HTML5:** Used exclusively for text markup and page layout.

---

## 📈 Learning Outcomes

Through this exercise, I practiced:
1. Nesting inline styling tags (like combining `<i>` and `<u>` inside headings).
2. Segmenting historical text into scannable sub-headings (`<h3>`).
3. Managing layout and spacing relying solely on HTML tags.

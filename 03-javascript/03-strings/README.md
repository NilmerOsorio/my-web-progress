# 🧵 String Methods in JavaScript

In this section, I learned how to work with **strings** in JavaScript using different built-in methods and properties. Since strings are used to store and manipulate text, understanding how they work is essential for building dynamic and interactive applications.

## 📚 Topics Covered

### 🔠 `toUpperCase()` & `toLowerCase()`

These methods are used to convert the characters of a string to either uppercase or lowercase. They are especially useful when formatting text or comparing user input without worrying about letter capitalization.

---

### 🔢 Accessing Characters with Indexes

Every character in a string has a numerical position called an **index**, starting at `0`. Using square brackets (`[]`), it is possible to access a specific character within a string.

---

### 📏 `length`

The `length` property returns the total number of characters in a string, including spaces. It is commonly used to validate input, count characters, or determine the size of a text.

---

### 🔍 `indexOf()`

The `indexOf()` method searches for a specific character or word within a string and returns its position. If the value can't be found, it returns `-1`.

---

### ✂️ `slice()`

The `slice()` method extracts a portion of a string based on a starting and optional ending index. It's useful when only part of a string is needed.

---

### 🔄 `replace()` & `replaceAll()`

The `replace()` method replaces only the first occurrence of a specified value, while `replaceAll()` replaces every occurrence found within the string. These methods are commonly used to modify or format text.

---

### 💬 Template Literals

Template literals are created using backticks (`` ` ``) instead of single or double quotes. They provide a cleaner and more readable way to create strings, especially when working with dynamic content.

---

### ✨ String Interpolation

String interpolation allows variables or expressions to be inserted directly into a string using the `${}` syntax inside a template literal. This approach is more readable and easier to maintain than traditional string concatenation.

---

These concepts provide a solid foundation for handling text in real-world JavaScript applications.

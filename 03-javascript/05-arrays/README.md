# 📚 JavaScript Arrays

In this section, I learned about **arrays in JavaScript** and how they are used to store and organize multiple values in a single variable. I practiced how to create arrays, access their elements, modify their content, and use different methods to manage data.

## Main Concepts

- An **array** is a collection of elements stored in a specific order.
- Arrays can store different types of data, such as strings, numbers, and booleans.
- There are different ways to create arrays, including using `[]` and the `Array` constructor.
- Array indexes start from **0**, meaning the first element is stored at index 0.
- JavaScript allows arrays to have empty positions when values are assigned to non-existing indexes.

## Common Methods and Properties

- `push()` adds new elements to the end of an array.
- `pop()` removes the last element from an array.
- `shift()` removes the first element from an array.
- `unshift()` adds elements to the beginning of an array.
- `length` returns the number of elements stored in an array and can also be used to clear its content.

## Copying and Modifying Arrays

- `slice()` creates a copy of a part of an array without changing the original array.
- `splice()` modifies the original array by adding, removing, or replacing elements.

## Key Takeaway

Arrays are one of the most important data structures in JavaScript because they allow developers to store, access, and manipulate collections of information in an organized way.

## 🔀 Conditional Statements in JavaScript

In this section, I learned how to use **conditional statements in JavaScript** to control the flow of a program. Conditionals allow applications to make decisions based on whether a specific condition is true or false, making programs more dynamic and capable of responding to different situations.

### 🔹 `if` Statement
The `if` statement is the basic conditional structure in JavaScript. It allows a block of code to be executed only when a specific condition is true. It is commonly used to perform actions when a certain requirement is met.

### 🔹 `else if` Statement
The `else if` statement allows a program to check multiple conditions. It is useful when there are different possible outcomes and the program needs to evaluate them in order until it finds a valid condition.

### 🔹 `else` Statement
The `else` statement defines a default action that will be executed when none of the previous conditions are true. It works as a fallback option to handle situations where no other condition applies.

### 🔹 Ternary Operator `?:`
The ternary operator is a shorter way to write simple `if...else` statements. It is useful for small decisions with only two possible outcomes, helping make the code more concise and readable.

### 🔹 `switch` Statement
The `switch` statement is used to handle multiple possible cases based on a single value. It provides a cleaner alternative to multiple `else if` statements when working with several predefined options.

By learning these conditional structures, I improved my understanding of decision-making in JavaScript and how to create programs that can react differently depending on specific conditions.

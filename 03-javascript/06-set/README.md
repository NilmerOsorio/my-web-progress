# 📚 JavaScript Sets

In this section, I learned about **Sets in JavaScript** and how they work as a data structure for storing unique values. I explored how to create Sets, modify their content, verify stored elements, and convert between Sets and arrays.

## Main Concepts

- A **Set** is a collection where every value must be unique.
- Unlike arrays, Sets automatically prevent duplicate values.
- Sets can store different types of data, such as strings, numbers, and booleans.
- They are created using the `Set` object.

## Methods and Properties

- `add()` allows adding new values to a Set.
- `delete()` removes a specific value.
- `has()` checks whether a value exists inside the Set.
- `size` returns the total amount of elements in the Set.

## Conversions

- A Set can be transformed into an array using `Array.from()`.
- An array can be converted into a Set using the `Set` constructor.

## Key Takeaway

Sets are useful when handling collections of data where duplicated values are not allowed. They provide a simple way to store and manage unique elements efficiently in JavaScript.

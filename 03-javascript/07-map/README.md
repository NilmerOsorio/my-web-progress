# 📚 JavaScript Maps

In this section, I learned about **Maps in JavaScript** and how they are used to store data as **key-value pairs**. I practiced how to create Maps, add and retrieve information, check stored values, and manage the data contained inside them.

## Main Concepts

- A **Map** is a data structure that stores information in pairs of **keys and values**.
- Each key is unique and is associated with a specific value.
- Maps can use different data types as keys.
- They are created using the `Map` object.
- Unlike regular objects, Maps maintain the order in which elements are added.

## Common Methods and Properties

- `set()` adds new key-value pairs or updates an existing key.
- `get()` retrieves the value associated with a specific key.
- `keys()` returns all the keys stored in the Map.
- `values()` returns all the values stored in the Map.
- `has()` checks if a specific key exists.
- `entries()` returns all key-value pairs from the Map.
- `size` returns the total number of elements.
- `delete()` removes a specific key and its associated value.
- `clear()` removes all elements from the Map.

## Key Takeaway

Maps are useful when working with structured data that requires a relationship between keys and values. They provide an efficient way to store, access, and manage information in JavaScript.

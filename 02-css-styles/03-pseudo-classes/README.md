# CSS Pseudo-Classes Practice 🎯

This is a project focused on practicing **CSS pseudo-classes**. 
It demonstrates how elements can change their style based on user interaction and position in the DOM.

---

## 📚 What This Project Covers

This exercise includes practical use of the most common CSS pseudo-classes:

- `:hover` → Styling elements when the mouse is over them.
- `:active` → Styling buttons while being clicked.
- `:focus` → Styling input fields when selected.
- `:first-child` → Styling the first element in a list.
- `:last-child` → Styling the last element in a list.
- `:nth-child(odd/even)` → Alternating styles in lists.
- `:not()` → Excluding specific elements from styling.
- `:visited` → Styling links that have been visited.

---

## 🧪 Features

- Interactive navigation index.
- Styled buttons with click effects.
- Input field with focus effects.
- Alternating list item styles.
- Hover effects for links.
- Basic layout using HTML and CSS only.

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 🎯 Learning Purpose

The goal of this project is to understand how CSS pseudo-classes work and how they can be used to create interactive and dynamic UI effects without JavaScript.

---

## ✨ What I Learned 

- How pseudo-classes react to user interaction.
- How CSS applies styles based on element state.
- How structural selectors like :nth-child() work.
- How to combine multiple selectors effectively.
